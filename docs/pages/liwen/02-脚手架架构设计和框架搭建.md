### lerna 文档
```
$ npm i lerna -g
$ lerna init
```
```
$ lerna create 
package name @mt-cli-dev/core
```
```
$ lerna add 
给包安装依赖(单个)
lerna add react packages/core = npm i react
```
```
$ lerna clean
删除所有包的 node_nodules 不删除packages.json 里的依赖目录
```
```
$ lerna bootstrap
类似于 $ mpm i？
```
```
$ lerna link 
本地包之间的依赖
```
```
## lerna 的 git 操作
lerna version
lerna changed
lerna diff
```
